# TransformMatrix::set_matrix_identity @ 0x44d210

Exact 19-instruction `tMatrix::Identity()` body. It resets the complete matrix,
including translation, while `set_matrix_rotation_identity` deliberately
preserves translation. Windows passes the matrix in `ECX`; Android independently
retains the authored method symbol.

The owner promotion also converts every proof-grade Windows caller to member
syntax. A legacy free-function declaration remains only as an ABI-compatible
surface for partial scratches whose source shape has not yet been revisited.
